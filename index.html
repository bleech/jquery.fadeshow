A simple fading slideshow. Takes an interval and an animation speed. Pauses on hover.

Getting Started

Download the production version or the development version.

Markup

<ul class="container">
    <li>Slide 1</li>
    <li>Slide 2</li>
    <li>Slide 3</li>
</ul>

Initialization

$('.container').fadeshow();

Options

// setting global default options
$.fn.fadeshow.options = {
  interval: 4000,
  speed: 600,
  onInit: function () {},
  onFade: function () {}
};

// setting custom instance options
$('.container').fadeshow({
  interval: 2000,
  speed: 1000,
  onInit: function () {},
  onFade: function () {}
});
